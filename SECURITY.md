# Security Policy

## Supported Versions
Currently, Project Parvati (Yugayu OS) is in active pre-release development. Only the latest commit on the `main` branch is actively supported for security updates.

## Reporting a Vulnerability
If you discover a vulnerability in the Zero-Trust IAM Gateway, cryptographic ledger, or node isolation protocols, **do not open a public issue.** Please email the core maintainers directly. We will provide a response within 48 hours and work with you to patch the vulnerability before public disclosure.
